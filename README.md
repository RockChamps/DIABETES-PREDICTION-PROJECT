# Diabetes Prediction using Machine Learning

This project builds a **Machine Learning model to predict diabetes** using medical attributes such as glucose level, blood pressure, BMI, insulin level, and age. The goal is to assist early detection of diabetes using predictive analytics.

## Dataset

The dataset includes medical features such as:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

## Models Used

Multiple machine learning algorithms were trained and evaluated:

* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest
* Gradient Boosting
* Naive Bayes
* Bagging Classifier

## Model Performance

Best model accuracy achieved: **~77%**
